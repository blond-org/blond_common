Welcome to BLonD Common’s documentation!¶

Contents¶

The Beam Longitudinal Dynamics code BLonD is a CERN software package for the simulation of longitudinal beam dynamics in synchrotrons.

Visit the BLonD website for more details and the github distribution to download the code.
